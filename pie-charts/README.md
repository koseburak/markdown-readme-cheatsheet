# Pie Chart Examples

## Example 1

### Code:
```
pie title NETFLIX
         "Time spent looking for movie" : 90
         "Time spent watching it" : 10
```

### Output:
::: mermaid
pie title NETFLIX
         "Time spent looking for movie" : 90
         "Time spent watching it" : 10
:::
